# DIO - Microsoft Azure Essentials

## Projeto 04 - Construindo Arquiteturas no Azure

Ao Selecionar "Brazil" em "Select your geography" aparecerá duas opções, com diferentes valores para
"Datacenter region", "Location", "Availabel to".

Tópicos:
- Criar grupos de recursos
- Criar grupo de acesso
- Ver logs de atividades
- IAM (Controle de acesso)
- Marcações
- Visualizador de recursos
- Eventos
- Criar rede virtual
